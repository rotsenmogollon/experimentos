# Calendario de publicaciones – Financiera Mi Casa

## Resumen

- **40 piezas** en la carpeta `Artes/`: 20 **posts** (1:1) y 20 **historias** (9:16).
- **Publicación:** 1 post + 1 historia por día, de **lunes a viernes**.
- **Inicio:** Miércoles 4 de marzo de 2026.
- **Duración:** 20 días laborables (4 semanas).

## Archivos generados

| Archivo | Descripción |
|--------|-------------|
| `calendario_financiera_mi_casa.csv` | Archivo de control con fecha, tipo, nombre de archivo, copy completo, CTA, web y contacto. |
| `Artes/*.png` | Imágenes renombradas con la nomenclatura `YYYYMMDD_POST_o_HISTORIA_tema_NN.png`. |
| `renombres.txt` | Listado original → nuevo (referencia). |
| `calendario_contenido.py` | Script usado para generar fechas, asignación y copies. |

## Uso del CSV

1. Abrir `calendario_financiera_mi_casa.csv` en Excel o Google Sheets.
2. Cada fila es una pieza (post o historia) con su **fecha**, **nombre de archivo** y **copy** listo para pegar.
3. **Contacto:** teléfono 6824-3794. La financiera no tiene sitio web; el copy solo incluye teléfono y dirección.
4. Opcional: añadir columnas *Programado* / *Publicado* para seguimiento.

## Nomenclatura de archivos

- `20260304_POST_auto_01.png` = 4 mar 2026, post, tema auto, pieza 1.
- `20260304_HISTORIA_casa_01.png` = 4 mar 2026, historia, tema casa, pieza 1.

Los archivos están ordenados por fecha; cada día tiene exactamente un post y una historia.

## Distribución de temas

- **Auto (préstamo con garantía de auto):** mayoría de piezas.
- **Casa / liquidez con propiedad:** varias piezas repartidas.
- **Ubicación / Bellavista / oficina:** piezas de invitación a visita y atención.

La asignación evita repetir el mismo tipo de mensaje dos días seguidos.
